# NPS

一款轻量级、高性能、功能强大的内网穿透软件。

# 特性

- 协议支持全面，兼容几乎所有常用协议
- Web 管理界面，方便配置和监控
- 客户端无需配置，直接运行即可使用
- 支持 TCP、UDP、HTTP、HTTPS 等多种协议
- 支持端口映射、域名解析、反向代理等功能

# 快速开始

## 服务端部署

### Linux/Mac

```bash
# 启动
sudo ./nps install
sudo nps start

# 或者直接运行
./nps
```

### Windows

```powershell
# 启动
nps.exe install
nps.exe start

# 或者直接运行
nps.exe
```

## 客户端部署

```bash
# 直接运行（推荐）
./npc -server=你的服务器地址:8024 -vkey=你的vkey

# 安装系统服务
./npc install -server=你的服务器地址:8024 -vkey=你的vkey
```

### Windows 客户端

```
双击 npc.exe，按照提示输入服务器地址和 vkey 即可完成配置。
```

# 配置说明

默认配置文件位于 `conf/nps.conf`，主要配置项：

| 配置项 | 说明 |
|--------|------|
| http_port | HTTP 端口 |
| https_port | HTTPS 端口 |
| bridge_port | 客户端连接端口 |
| vkey | 认证密钥 |

- 2025-11-05 v0.26.27
  新增：
  - 1.客户端新增【IP白名单】，可通过两种方式添加IP:
    - 方式1：在【客户端】新建/修改页面，配置IP授权密码，不在IP白名单的外网访问将被跳转到 IP 授权页面，输入正确的授权密码，即可完成添加 IP。
