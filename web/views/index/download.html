使用说明

命令行客户端
  1. 下载对应平台的压缩包并解压
  2. 在终端/命令行中运行： 
    ./npc -server={{.ip}}:{{.p}} -vkey=你的密钥
  3. 密钥可在"客户端管理"中创建客户端后获取
图形界面客户端
  1. 下载对应平台的压缩包并解压
  2. 双击运行程序，自动打开浏览器
  3. 在 Web 界面填写服务器地址和密钥
  4. 点击"启动连接"即可

macOS 用户：首次运行会请求管理员权限，用于安装系统服务功能
