客户端下载

请根据您的操作系统和架构下载 nps 客户端。我们提供命令行 (CLI) 和图形用户界面 (GUI) 两种版本（如果可用）。

Linux 客户端

架构 CLI 客户端 GUI 客户端
amd64 下载 暂无
arm64 下载 暂无
安装指南:

1. 下载适用于您架构的客户端。
2. 解压文件：tar -zxvf linux_amd64_client.tar.gz (请替换为实际下载的文件名)。
3. 运行客户端：./npc (命令行版) 或 ./npc_gui (图形界面版，如果可用)。
