客户端下载与安装

欢迎来到 nps 客户端下载页面。请根据您的操作系统和处理器架构选择合适的客户端版本。我们提供两种类型的客户端：

  • 命令行客户端 (CLI): 轻量级，适合服务器或熟悉命令行的用户。
  • 图形用户界面客户端 (GUI): 提供直观的操作界面，适合桌面用户（目前仅支持 Windows 和 macOS）。

下载后，请参考下方详细的安装指南。

Linux 客户端下载与安装

适用于 Linux 系统的 nps 命令行客户端。目前暂无官方 GUI 版本。

架构 CLI 客户端
amd64 下载 (amd64)
arm64 下载 (arm64)
安装指南 (CLI):
  1. 下载客户端：

    根据您的 Linux 系统架构（通常是 amd64 或 arm64），点击上方表格中对应的“下载”链接，将客户端压缩包保存到您的计算机。

  2. 解压客户端：

    打开终端（Terminal），导航到您下载文件的目录（例如：cd ~/Downloads）。然后使用 tar 命令解压文件。请将 linux_amd64_client.tar.gz 替换为您实际下载的文件名：

    tar -zxvf linux_amd64_client.tar.gz

    解压后，您会得到一个名为 npc 的可执行文件。

  3. 运行客户端：

    首先，赋予 npc 可执行权限：

    chmod +x npc

    然后，直接运行客户端：

    ./npc

    首次运行会提示您输入服务器地址和验证密钥。按照提示操作即可。

    （可选）后台运行：如果您希望客户端在后台持续运行，可以使用 nohup 命令：

    nohup ./npc &

    要停止后台运行的客户端，您需要查找其进程 ID (PID) 并使用 kill 命令。例如：ps aux | grep npc 找到 PID，然后 kill <PID>。

    （可选）安装为系统服务：对于长期稳定运行，强烈建议将其安装为系统服务。具体步骤请参考官方文档。

Windows 客户端下载与安装

适用于 Windows 系统的 nps 命令行和图形界面客户端。

架构 CLI 客户端 GUI 客户端
amd64 下载 (amd64) 下载 GUI (amd64)
arm64 下载 (arm64) 下载 GUI (arm64)
安装指南 (CLI):
  1. 下载客户端：

    根据您的 Windows 系统架构，点击上方表格中对应的“下载”链接，将客户端 ZIP 压缩包保存到您的计算机。

  2. 解压客户端：

    找到下载的 ZIP 文件，右键点击选择“全部提取”或使用您喜欢的解压工具将其解压到一个您方便的目录（例如：C:\nps-client）。解压后，您会得到一个名为 npc.exe 的可执行文件。

  3. 运行客户端：

    打开命令提示符 (CMD) 或 PowerShell，导航到您解压客户端的目录：

    cd C:\nps-client

    然后，运行客户端：

    npc.exe

    首次运行会提示您输入服务器地址和验证密钥。按照提示操作即可。

    （可选）后台运行：您可以创建一个批处理文件 (.bat) 或使用任务计划程序来使其在后台运行。

    （可选）安装为系统服务：对于长期稳定运行，建议将其安装为系统服务。具体步骤请参考官方文档。

安装指南 (GUI):
  1. 下载客户端：

    根据您的 Windows 系统架构，点击上方表格中对应的“下载 GUI”链接，将客户端 ZIP 压缩包保存到您的计算机。

  2. 解压客户端：

    找到下载的 ZIP 文件，右键点击选择“全部提取”或使用您喜欢的解压工具将其解压到一个您方便的目录。解压后，您会得到一个名为 npc_gui.exe 的可执行文件。

  3. 运行客户端：

    双击 npc_gui.exe 即可运行图形界面客户端。按照界面提示进行配置和连接。
