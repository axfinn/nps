Linux 客户端下载

适用于 Linux 系统的 nps 命令行客户端。目前暂无官方 GUI 版本。

架构 CLI 客户端
amd64 (64位) 下载 (amd64)
arm64 (ARM 64位) 下载 (arm64)
Linux 安装与运行指南 (CLI):
1. 下载客户端

根据您的 Linux 系统处理器架构（通常是 amd64 或 arm64），点击上方表格中对应的“下载”按钮，将客户端压缩包保存到您的计算机（例如，下载到 ~/Downloads 目录）。

2. 解压客户端

打开终端（Terminal），导航到您下载文件的目录。然后使用 tar 命令解压文件。请将 linux_amd64_client.tar.gz 替换为您实际下载的文件名：

cd ~/Downloads
tar -zxvf linux_amd64_client.tar.gz

解压后，您会得到一个名为 npc 的可执行文件。

3. 赋予执行权限

为了能够运行 npc，您需要为其添加执行权限：

chmod +x npc
4. 运行客户端

直接在终端中运行客户端：

./npc

首次运行 npc 时，它会提示您输入 nps 服务器的地址（例如：your_server_ip:8024）和验证密钥（vkey）。请按照提示输入正确的信息。

提示：您也可以通过配置文件来启动客户端，具体请参考官方文档。

5. （可选）后台运行客户端

如果您希望 npc 客户端在关闭终端后仍然在后台持续运行，可以使用 nohup 命令：

nohup ./npc &

要停止后台运行的客户端，您需要先查找其进程 ID (PID)，然后使用 kill 命令。例如：

ps aux | grep npc
kill <PID>
6. （可选）安装为系统服务

对于需要长期稳定运行的场景，强烈建议将 nps 客户端安装为系统服务（如 Systemd 服务）。这样可以确保客户端在系统启动时自动运行，并在意外停止后自动重启。具体安装步骤和配置方法，请参考 nps 官方文档中关于“注册到系统服务”的部分。

Windows 客户端下载

适用于 Windows 系统的 nps 命令行 (CLI) 和图形用户界面 (GUI) 客户端。

架构 CLI 客户端 GUI 客户端
amd64 (64位) 下载 (amd64) 下载 GUI (amd64)
arm64 (ARM 64位) 下载 (arm64) 下载 GUI (arm64)
Windows 安装与运行指南 (CLI):
1. 下载客户端

根据您的 Windows 系统处理器架构，点击上方表格中对应的“下载”按钮，将客户端 ZIP 压缩包保存到您的计算机。

2. 解压客户端

找到下载的 ZIP 文件，右键点击选择“全部提取”或使用您常用的解压工具（如 WinRAR, 7-Zip）将其解压到一个您方便的目录（例如：C: ps-client）。解压后，您会得到一个名为 npc.exe 的可执行文件。

3. 运行客户端

打开命令提示符 (CMD) 或 PowerShell，导航到您解压客户端的目录：

cd C:\nps-client

然后，运行客户端：

npc.exe

首次运行 npc.exe 时，它会提示您输入 nps 服务器的地址和验证密钥。请按照提示输入正确的信息。

提示：您也可以通过配置文件来启动客户端，具体请参考官方文档。

4. （可选）后台运行客户端

在 Windows 上，您可以通过创建批处理文件 (.bat) 或使用任务计划程序来使其在后台运行。例如，创建一个 start_npc.bat 文件，内容为 start /B npc.exe，然后运行此批处理文件。

5. （可选）安装为系统服务

对于需要长期稳定运行的场景，建议将 nps 客户端安装为 Windows 服务。这样可以确保客户端在系统启动时自动运行，并在意外停止后自动重启。具体安装步骤和配置方法，请参考 nps 官方文档中关于“注册到系统服务”的部分。

Windows 安装与运行指南 (GUI):
1. 下载客户端

根据您的 Windows 系统处理器架构，点击上方表格中对应的“下载 GUI”按钮，将客户端 ZIP 压缩包保存到您的计算机。

2. 解压客户端

找到下载的 ZIP 文件，右键点击选择“全部提取”或使用您常用的解压工具将其解压到一个您方便的目录。解压后，您会得到一个名为 npc_gui.exe 的可执行文件。

3. 运行客户端

双击 npc_gui.exe 即可运行图形界面客户端。首次运行会引导您进行服务器地址和验证密钥的配置。按照界面提示进行操作即可。
